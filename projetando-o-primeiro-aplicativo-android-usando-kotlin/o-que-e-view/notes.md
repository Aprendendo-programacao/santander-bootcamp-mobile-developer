# O que é uma View

### Views

* Todo componente é uma **View**

* Toda View é clicável

* TAG: `<nomeDoComponente>`

* 3 elementos básicos: id, layout_width, layout_height

* **Views**  herdam métodos, aparência, estilo e atributos de componentes (botão, imagens, texto e etc) ou pode ser criados de maneira customizada

* **Exemplos de Views**

  * CheckBox (herda de Button)

  * RadioButton (herda de Button)

  * AutoCompleteTextView (herda de EditText)
