# Componente Activity

### Activity

* Um dos componentes principais de um projeto Android

* Utilização de instâncias dos métodos *callback* de ciclos de vida (em vez de uma única função *main*)

* Pode corresponder a uma ou mais telas

* Toda *activity* deve ser registrada no *manifest* (Android Studio faz isso automaticamente)

* O *layout da activity* define o que será exibido para o usuário

* Responsável por gerenciar os ciclos de vida, interações com o usuário, abertura de outras telas e aplicações ([***intent***](https://developer.android.com/guide/components/intents-filters?hl=pt-br)), etc ...
