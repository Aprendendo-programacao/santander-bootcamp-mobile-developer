# Configuração inicial do projeto

### Configurações do projeto

* Criando um projeto de **módulo único**, mas é possível criar projetos com multi-módulos

* Selecionar a versão mínima do Android

* Arquivos gerados:

  ![](https://developer.android.com/images/tools/projectview-p2.png)

### Manifest

* Informações essenciais para as ferramentas de compilação do Android, para o Sistema Operacional Android e a Google Play

* Armazena a declaração de todas as activities, permissões do app, etc...

* Arquivo gerado e escrito em XML

* Declaração por TAGs

### API Level

* APIs que tiveram grandes mudanças: 18, 21, 24 e 39

* APIs recomendadas: 19 até 24
