# A relação entre Kotlin e Android

### Vantagens de usar Kotlin

* Segura

* Estruturada

* Menos verboso

* Mais otimizado

* Grande relevância no mercado

* Interoperabilidade com o Java

* _Smart casts_

* _Null safety_

* Android é _Kotlin first_

### Onde surgiu a relação do Kotlin com Android

* **2016**: 1º versão estável do Kotlin é liberada

* **2017**: Google anuncia suporte a Kotlin para desenvolvimento Android

* **2018**: 2º linguagem preferida dos desenvolvedores

* **2019**: Google anuncia que Kotlin é a nova linguagem oficial para desenvolvimento Android (_Kotlin first_)
