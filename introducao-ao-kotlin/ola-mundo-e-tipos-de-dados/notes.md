# Olá mundo e Tipos de dados

### Print "Hello World"

```kotlin
fun main() {
    println("Hello world")
}
```

### Tipos de dado

* **Int** (Valor máximo: `Int.MAX_VALUE`)
* **Long** (Valor máximo: `Long.MAX_VALUE`)
* **Float** (Valor máximo: `Float.MAX_VALUE`)
* **Double** (Valor máximo: `Double.MAX_VALUE`)
* **Array**
* **Boolean**
* **Char**
* **Byte** (Valor máximo: `Byte.MAX_VALUE`)
* **Short**
* **Null**

### Conversão 

* `.toByte()`
* `.toShort()`
* `.toInt()`
* `.toLong()`
* `.toFloat()`
* `.toDouble()`
* `.toChar()`
