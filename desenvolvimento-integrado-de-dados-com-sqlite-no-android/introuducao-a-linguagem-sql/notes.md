# Introdução a linguagem SQL

### Definição de banco de dados

* Armazenar e gerenciar, de forma segura, os dados

### Definição linguagem SQL

* SQL = Structured Query Language

* Responsável por dar instruções ao banco de dados

### Definição de *Query*

* É qualquer comando, em SQL, que será realizado em um banco de dados
