# Práticas recomendadas e dicas de segurança

# Práticas recomendadas para segurança de aplicativos

* Aplicar comunicação segura (integridade das dados e dispositivos)

* Oferecer as permissões corretas (apenas as permissões necessários para o funcionamento do aplicativo)

* Armazenar dados de forma segura

* Manter os serviços e as dependências atualizados

### Dicas de segurança

* Armazenamento de dados

  * Usar o armazenamento interno
  * Usar o armazenamento externo (não recomendado para armazenamento de dados sensíveis)
  * Usar provedores de conteúdo

* Usar permissões
