# Android Debug Bridge

### Funções

* Operações que facilitam a manipulação de aplicação no modo *debug*

* Pode ser chamada pelo terminal, mas possui plugin com principais funcionalidades (ADB Idea)

### ADB Idea

> Página oficial: [clique aqui](https://plugins.jetbrains.com/plugin/7380-adb-idea)

* Abrir o menu do ADB Idea: `Tools > ADB Idea > ADB Operations Popup...` ou `CTRL + ALT + SHIFT + A`
